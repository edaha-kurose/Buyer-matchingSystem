## 1. 御社が抱える「提案受付」の課題

### ⏰ 深刻な時間拘束

| 項目 | 数値 |
|------|------|
| 1件あたりの所要時間 | 30分〜1時間 |
| 年間プレゼン受付数 | 数十〜数百件 |
| **年間拘束時間** | **150〜375時間/人** |

### 📉 低い採用率

- 提案の **85%は不採用**
- 大半の時間が「断る」ための時間に消費
- 良い提案を見逃すリスク

### 😓 判断精度の低下

- 疲労による集中力低下
- 担当者による評価のばらつき
- 比較検討の時間不足

### 💸 高コスト構造

| 項目 | コスト |
|------|--------|
| バイヤー人件費（45分） | ¥5,625 |
| 事前準備・事後処理 | ¥1,875 |
| 会議室・設備費 | ¥500 |
| **1件あたり合計** | **¥8,000** |
| **年間300件の場合** | **¥2,400,000** |

## 2. 解決アプローチ

### 💡 コンセプト

> **「85%の不採用案件をAIが処理し、バイヤーは15%の採用候補に集中する」**

### 🔄 処理フロー

```
📄 資料提出 → 🤖 AI解析 → ❓ 自動Q&A → 📊 スコアリング → 👤 人間が判断
```

| ステップ | 内容 |
|----------|------|
| 1. 資料提出 | サプライヤーが資料をアップロード |
| 2. AI解析 | 内容を自動抽出、条件と照合 |
| 3. 自動Q&A | AIが不足情報を自動で質問 |
| 4. スコアリング | 条件適合度を自動評価 |
| 5. 人間が判断 | 採用候補のみバイヤーがレビュー |

## 4. 要約おまとめ機能（NEW）

### 🔍 AIファクトチェック付き要約

サプライヤーからの提案を自動的に要約し、**客観的なファクトチェック**を実施した上で、重要ポイントを抽出します。

### 📋 機能の流れ

```
提案資料 → 情報抽出 → ファクトチェック → 要約レポート生成
```

### ✅ ファクトチェック項目

| カテゴリ | チェック内容 |
|----------|-------------|
| 企業情報 | 法人登記情報、設立年、資本金の照合 |
| 数値データ | 市場規模、シェア率の妥当性検証 |
| 実績・事例 | 導入実績、特許番号の確認 |
| 表現 | 誇大表現、根拠のない主張の検出 |

### 📊 信頼度スコア

| ランク | スコア | 意味 |
|--------|--------|------|
| 🟢 A | 85-100点 | 高信頼：データ検証済み |
| 🟡 B | 70-84点 | 中信頼：一部要確認 |
| 🟠 C | 50-69点 | 低信頼：複数の確認事項あり |
| 🔴 D | 0-49点 | 要注意：重大な懸念あり |

### 📄 要約レポート例

```
┌─────────────────────────────────────────────────────┐
│ 📋 ABC株式会社 - 新型IoTセンサーの提案              │
├─────────────────────────────────────────────────────┤
│                                                     │
│ 📝 3行サマリー                                      │
│ 製造現場向けIoTセンサーの提案。従来比30%の          │
│ コスト削減を実現。大手自動車メーカー3社への         │
│ 導入実績あり。最小ロット100個から対応可能。         │
│                                                     │
│ ┌───────────────┐  ┌───────────────┐              │
│ │ 信頼度: 🟢 87点│  │ 推奨度: A     │              │
│ └───────────────┘  └───────────────┘              │
│                                                     │
│ 🎯 重要ポイント                                     │
│ ✅ 価格競争力: 30%削減（検証済み）                 │
│ ✅ 品質: ISO9001認証取得（確認済み）               │
│ ⚠️ 導入実績: 2社は確認済み、1社は未確認           │
│                                                     │
│ 💬 確認推奨事項                                     │
│ • 未確認の導入実績1社について詳細確認              │
│ • 繁忙期の納期リスクを確認                         │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 💡 メリット

- **詳しく聞くべき提案かどうかを即座に判断**
- **客観的データに基づく信頼性評価**
- **確認すべきポイントが明確**
- **3行サマリーで概要を瞬時に把握**
